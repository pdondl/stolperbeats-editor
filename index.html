How to use
Load / Save: Load a Stolperbeats .txt pattern bank file, edit patterns, then save back to .txt for loading onto the hardware via SD card.
Patterns: Click the numbered buttons to switch between the 24 patterns. Buttons with a green underline contain data.
Steps: Click a step to toggle it on/off. Kick and Snare have 64 steps (shown as 4 rows of 16); HiHat and Perc tracks have 16 steps.
Track length: Kick and Snare tracks can be set to shorter lengths (16/32/48) — extra steps are dimmed and the pattern loops at the set length.
Samples: Drag & drop a .wav file onto a track label to assign a custom sample for preview. Samples are stored in the browser and persist across sessions.
Mute: Click the M button next to a track to mute it during playback.
Choke: HiHat1 and HiHat2 share a choke group — triggering one cuts off the other.
Copy / Paste: Copy the current pattern and paste it into another pattern slot.
Custom samples
By default, each track uses a built-in synth voice. To use your own sounds:
1. Drag & drop a .wav file onto the track label (e.g. “KICK”) on the left. The label turns green and shows the filename.
2. Samples are stored in the browser (IndexedDB) and persist across sessions — no need to re-load them every time.
3. To remove a custom sample and revert to the built-in synth, right-click the track label.
Mono .wav files at 44.1 kHz work best. Any sample rate or channel count will be converted automatically, but short one-shots give the tightest results.
Shuffle parameters
Shuffle (4–9): selects which of the 6 Stolperbeats shuffle patterns to use. Each rearranges the micro-timing of the 4 substeps within a beat.
Timing: Straight, Trip, Shake, Push, Clave — sets the rhythmic feel (triplet, swing, push, clave groove).
Feeling: Tight, Rolls, Flam, Slop — controls how precisely hits land on the grid.
