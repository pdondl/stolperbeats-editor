# Tripwire

Browser-based pattern editor for the [Stolperbeats](https://makingsoundmachines.com/stolperbeats/) Eurorack drum sequencer by Making Sound Machines.

Edit pattern banks, preview with shuffle and timing parameters, and save `.txt` files for loading onto the hardware.

**Live editor:** [plentyofnames.github.io/tripwire](https://plentyofnames.github.io/tripwire/)

Built by a human and a Claude.
